My Projects

Here are examples of my notable works:

E-Commerce Platform

Developed a fully responsive e-commerce platform for Henki Limited, a leading electronics retailer using HTML5 and CSS3 for responsive layouts, and SQL for efficient database management.

The website features a modern product catalog with advanced filtering, real-time inventory tracking, and secure payment processing. Implemented a user-friendly interface showcasing various electronic categories including smartphones, laptops, gaming consoles, and accessories.
The platform includes a robust search function, detailed product descriptions, customer reviews, and a streamlined checkout process. The implementation resulted in a 40% increase in online sales and significantly improved user engagement metrics.

portfolio Website

Crafted an elegant portfolio website for a professional photographer/videographer using HTML5, Tailwind CSS, and JavaScript.

The site features a stunning gallery with lazy loading for optimal performance, showcasing both photography and video content in a masonry grid layout. Implemented smooth transitions and animations to enhance the viewing experience of high-resolution images and video showreels.
The website includes a dynamic filtering system allowing visitors to sort through different photography categories such as weddings, portraits, and commercial work. Built a custom lightbox feature for immersive full-screen image viewing and integrated a video player for seamless showreel playback. Additionally, incorporated a contact form for client inquiries.

Football Preview Website

Engineered a dynamic football preview website using React, Node.js, and a SQL database, delivering real-time match statistics and team news.

The platform features an interactive dashboard displaying upcoming fixtures and team form guides. Implemented a live score tracking system, player performance metrics, and injury updates across major leagues. The website includes a responsive design optimized for both desktop and mobile viewing, ensuring fans can access match previews on any device.
Integrated an API to pull historical match data and statistical analysis, enabling users to make informed predictions. The implementation led to significant user engagement with over 10,000 monthly active users demonstrating its value as a comprehensive resource for football enthusiasts.

Java Chat Application

Developed a robust terminal-based chat application using Java and Java.net, enabling real-time communication between multiple users through a command-line interface.

The application implements a client-server architecture using TCP/IP sockets for reliable message delivery and synchronous communication. Built features including group chat capabilities, and user status updates. Implemented multi-threading to handle concurrent user connections, ensuring smooth message broadcasting and individual chat sessions.
Enhanced the system with features such as message timestamps, online user listing, and chat history storage. The clean terminal interface provides an efficient, lightweight communication tool that demonstrates core networking principles and Java's capabilities in building networked applications.
